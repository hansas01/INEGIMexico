Run the inegiReader.js in node.js from the terminal. The output is a CSV file with clean data for marketing. 

The commit includes CSV file for practice, change the filename in the code where indicated in inegiReader.js.

Go to https://inegi.org.mx/app/descarga/default.html to download data that you want to sort. 

Very useful for sales campaigns where you want to exclude businesses without phone and email listed. Ready to go for mailchimp or similar.
